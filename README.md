# Wine Quality Prediction

## Overview
This project predicts the quality of red wine using Machine Learning.

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS

## Machine Learning Models
- Decision Tree
- Random Forest (Best Model)
- XGBoost
